# Web Apps & Services

A series of three projects exploring the fundamentals of building and consuming REST APIs, progressing from a Node.js backend to a vanilla JS frontend to a full React/Next.js rebuild.

## What This Demonstrates

- Designing and implementing a **RESTful API** with Express, MongoDB, and Mongoose
- Consuming APIs on the client-side with the **Fetch API** and dynamic DOM manipulation
- Rebuilding a vanilla JS app as a **React/Next.js** component-based architecture
- **SWR** data fetching with a globally configured fetcher
- **Next.js** dynamic routing, layouts, and `getStaticProps` pre-rendering
- Client-side **pagination** and **search/filter** functionality
- Working with **React-Bootstrap** for responsive UI components

## Tech Stack

JavaScript · React · Next.js · Node.js · Express · MongoDB Atlas · Mongoose · SWR · Bootstrap · Git

## Project Progression

These three projects follow a **data-first approach**: build an API (1), consume it with vanilla JavaScript (2), then rebuild the frontend in React/Next.js (3). This mirrors a real-world scenario where a backend exists and the frontend evolves from a prototype to a modern framework.
